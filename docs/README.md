# Documentación BalanzaLog

Esta carpeta reúne notas breves para entender el equipo sin convertir el repositorio en un manual largo.

## Archivos

- `01_descripcion_general.md`: resumen del equipo, arquitectura y modos de operación.
- `02_guia_rapida.md`: uso básico, ejecución en Raspberry Pi y ubicación de datos.
- `img/`: carpeta sugerida para fotos del diseño 3D, montaje físico y pruebas.

## Imágenes sugeridas

Para que el README se vea mejor en GitHub, conviene agregar 2 o 3 imágenes:

```text
docs/img/balanzalog_render_3d.png
docs/img/balanzalog_equipo_fisico.jpg
docs/img/balanzalog_prueba_pesaje.jpg
```

Luego se pueden insertar en el README con:

```md
![BalanzaLog](docs/img/balanzalog_equipo_fisico.jpg)
```
